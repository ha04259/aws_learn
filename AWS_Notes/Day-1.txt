5. Cloud Service Models
Infrastructure as a Service (IaaS) --> example: EC2
 Responsible for 
 - Compute
 - Network
 - NW Security
 - Storage
 
Platform as a Service (PaaS) --> example: RDS
 Responsible for 
 - Compute
 - Network
 - NW Security
 - Storage
 - Runtime
 
Software as a Service (SaaS) --> example: Dropbox, G-Drive, MSAAS(IKS)
 Responsible for 
 - Compute
 - Network
 - NW Security
 - Storage
 - Runtime
